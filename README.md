## Running the site locally

To clone the repository and run the documentation site on your local machine,
follow these steps:

```bash
git clone https://github.com/ayoisaiah/f2-docs/
```

```bash
cd f2-docs
```

```bash
npm install
```

```bash
npm run docs:dev
```
